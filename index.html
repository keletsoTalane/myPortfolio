    Campus Navigator

    CampusNavigator is a robust web application designed specifically for the Tshwane University of Technology Polokwane campus, aimed at helping students navigate the campus efficiently. This project provides a comprehensive solution for locating lecture halls, administrative offices, libraries, and other key campus locations. It enhances the campus experience through its user-friendly interface, detailed maps, and real-time navigation features. Technologies Used: Frontend: HTML, PHP, JavaScript (including TypeScript for mapping implementation), CSS Backend: PHP for server-side scripting Database: MySQL for storing location and user data Framework: Bootstrap for responsive design and layout Mapping Integration: MappedIn for configuring the interactive campus map

    GBV-Support-Network

    GBV Support Network is a dedicated web application providing essential support and resources for survivors of gender-based violence (GBV). It offers a safe and supportive environment where victims can share their experiences, seek assistance, and access advice. The platform facilitates community-building among users by allowing them to create or join groups based on their shared experiences, fostering solidarity and mutual support. Technologies Used: Backend: Python with Django framework for server-side logic and RESTful API development. Database: MySQL for storing user profiles, group information, and posts securely. Frontend: HTML, JavaScript, and CSS for user interface design and interactivity. Framework: Bootstrap for responsive web design, ensuring compatibility across devices and browsers.
